Instructions
Please! bear in mind the following instructions in order to use the calculator properly
  1. Enter an expression and then press equal to get the answer.
  2. To calculate Sine, Cosine, Tangent, Cotangent, Secant, Cosecant, Square root, Cube root, Press the corresponding button, then enter a number and a closing parenthesis
  3. Click ↩ to delete last one element.
  4. Click Ac to clear the output field.
Remember! You can also input through Keyboard.
Keyboard input is given below.
Keyboard Input
  • delete Clear output field.
  • backspace Delete one last element.
  • enter Calculate expression.
  • numpadNumbers and operators input.
  • top rowNumbers input
